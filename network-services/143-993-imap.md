# 143,993 - IMAP

### Service enumeration

#### Scan

```sh
nmap <IP> -p143,993 -sC  -sV
```

#### Service interaction

```sh
nc -nv <IP> 143

# If SSL/TLS
openssl s_client -connect <IP>:imaps
```

#### Syntax

```sh
# Authenticate to IMAP
1 LOGIN <username> <password>

# List inboxes
A1 list "" "*"

# Select the inbox
g21 SELECT "<INBOX>"

# Retreive the entire message
F1 fetch <MESSAGE_ID> RFC822
```
